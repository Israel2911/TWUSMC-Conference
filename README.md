# TWUSMC-Conference

This repository contains the official website files for the 11th Annual Leadership Conference co-hosted by Stella Amaris College and Trinity Western University.

## Contents
- **index.html**: Main landing page code (conference agenda, live time/weather, interactive globe, brochure download).
- **IMG_7472.jpeg**: Cover image for the conference feature book, "The Fifth Intelligence".
- **GLOB-SMC-Leadership-Conference-2025_Brochure_Final5.pdf**: Downloadable conference brochure for attendees and partners.
- **image.jpg**: Additional images/as needed (replace in HTML if used).

## How to Preview Locally
1. Clone or download this repo.
2. Ensure all files are in the same directory.
3. Double-click `index.html` to open your conference page in a web browser.

## How to Deploy
- **Render.com**: Connect your GitHub repository and create a Static Site. Site will auto-update with pushes.
- **GitHub Pages**: Go to your repo Settings > Pages, select Branch=root and publish. Access via the GitHub Pages URL.

## Customization
- For additional sections, update or expand `index.html`.
- Add more images or assets by uploading to this repo and updating references in the HTML code.
- For interactive features (forms, speaker database), code new pages and link from the navigation bar.

For help or questions, contact the site maintainer or organizing committee.
